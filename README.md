# 算術勇者クエスト ～AIの塔～

ドラクエ風のドット絵 ×「風来のシレン」風のローグライク数学RPG。
主人公は **AI（ロボット）**。塔を **1フロアずつ上に登り**、立ちはだかる
モンスターを **数学の問題** で倒していきます。**5階ごとにボス**が出現！

## あそびかた

1. フィールド（1画面＝1フロア）を **方向キー / WASD**（スマホは画面の十字キー）で移動。
2. **敵にぶつかると数学バトル**。答えを入力して **Enter**（スマホは数字パッド＋こうげき！）。
   - 正解 → 敵にダメージ ／ 連続正解で **🔥コンボ**（EXPボーナス）
   - 不正解・時間切れ → 自分がダメージ、コンボはリセット
3. 敵を倒すと EXP → **レベルアップで HP全回復＆最大HPアップ**。
4. **🧪ポーション**を踏むと HP回復。
5. **▲階段** にのると **次の階へ**。
6. **5階ごとにボス**。ボスは複数回正解しないと倒せず、倒すと階段が出現。
7. 倒れたら、その回の到達階＆**最高到達階（記録）**が表示されます。

## 特徴

- 🏯 **タワー型ダンジョン**：1画面完結のフロアを上へ上へ（無限に深くなる）
- 👾 **敵がフロア上に見える**シレン風。敵は8種＋ボス4種
  - スライム / こうもり / おおねずみ / ゴースト / メカむし / ゴーレム / よろいへい / ドラゴン
  - ボス：キングスライム / がいこつナイト / じごくドラゴン / まおう
- ➗ 階が深いほど問題が難化（たし算 → かけ算 → わり算 → 方程式 → 2桁計算）
- 🔥 コンボ・🧪回復アイテム・🏆最高到達階の記録（ブラウザに保存）
- 🎨 5階ごとにダンジョンの色（ゾーン）が変化
- 🖼 すべてCanvasに手描きしたドット絵・外部画像なし
