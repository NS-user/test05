# 算術勇者クエスト ～AIの冒険～

ドラクエ風のドット絵RPG。主人公は **AI（ロボット）**。
モンスターとのバトルは **数学の問題**！ 正解すると攻撃でき、敵を倒すと
経験値が手に入って **レベルアップ** します。

![hero](https://img.shields.io/badge/HERO-AI-blue) ![type](https://img.shields.io/badge/battle-MATH-yellow)

## あそびかた

1. `index.html` をブラウザで開くだけ（ビルド不要）。
2. タイトルで「ぼうけんをはじめる」。
3. フィールドを **方向キー / WASD** で移動。
4. **濃い草むら** を歩くとモンスターが出現！
5. 出題された数学の問題の答えを入力して **Enter**（または「こうげき！」）。
   - 正解 → 敵にダメージ
   - 不正解・時間切れ → 自分がダメージ
6. 敵を倒すと EXP 獲得 → レベルアップで HP 全回復＆最大HPアップ。

## 特徴

- 🤖 主人公はドット絵のAIロボット
- 👾 4種類のモンスター（スライム / こうもり / ゴーレム / ドラゴン）
- ➗ レベルに応じて問題が難化
  - Lv1-2: たし算・ひき算
  - Lv3-4: かけ算
  - Lv5-6: わり算・複合計算
  - Lv7-9: かんたんな方程式・2乗
  - Lv10+: 1次方程式・2桁のかけ算
- ⏱ 強い敵ほど制限時間は長いが、間違えたときのダメージも大きい
- 🎨 すべてCanvasで描いた手描きドット絵・外部画像なし
