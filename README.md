# まなびクエスト ～知識の塔～（学習RPG）

ドラクエ風ドット絵 ×「風来のシレン」風ローグライクの **学習RPG**。
塔を **1フロアずつ上に登り**、立ちはだかるモンスターを **○×クイズ** で倒していきます。
**科目を選んで** プレイでき、**5階ごとにボス**が出現！

**サンプル科目＝司法試験／予備試験（短答）対策**。憲法・民法・刑法・民訴・刑訴・
商法会社法・行政法の○×問題を、**解説つき**で出題します（おまけで「算数」も収録）。

## あそびかた

1. タイトルで「とうに のぼる」→ **科目を選ぶ**。
2. フィールド（1画面＝1フロア）を **方向キー / WASD**（スマホは画面の十字キー）で移動。
3. **敵にぶつかると○×バトル**。記述が正しければ **◯**、誤っていれば **✕**。
   - キーボード：◯=`O`/`1`/`←`、✕=`X`/`0`/`→`、つぎへ=`Enter`
   - 正解 → 敵にダメージ ／ 連続正解で **🔥コンボ**
   - 不正解・時間切れ → 自分がダメージ、コンボはリセット
4. 回答後に **解説** が出るので、読んでから「つぎへ ▶」。
5. 敵を倒すと EXP → **レベルアップで HP全回復＆最大HPアップ**。🧪ポーションでも回復。
6. **▲階段** にのると次の階へ。**5階ごとにボス**（複数回正解で撃破→階段出現）。
7. 倒れると、到達階・最高到達階に加えて、その回の **正答率** が表示されます。

## 科目を増やすには

問題は `questions.js` にデータとして分離してあります。

- `window.SUBJECTS` … 科目の一覧（選択画面の並び）
- `window.QUESTIONS` … `{ s:科目キー, q:記述, a:正誤(true=正しい), e:解説 }` の配列

新しい科目を `SUBJECTS` に1行足し、その科目キーの問題を `QUESTIONS` に追加するだけで、
他の試験・教科にも展開できます（`type:'gen'` は算数のような動的生成科目）。

## 特徴

- 📘 **科目選択式**：司法試験7科目＋全科目ミックス＋おまけ算数
- 📝 **解説つき○×**：答えるたびに根拠条文・ポイントを表示（学習向け）
- 📊 **正答率トラッキング**：HUDとゲームオーバー画面に表示
- 🏯 **タワー型ダンジョン**：1画面完結のフロアを上へ上へ（無限）
- 🔥 コンボ・🧪回復アイテム・🏆最高到達階の記録（ブラウザに保存）
- 🖼 すべてCanvasに手描きしたドット絵・外部画像なし
